### 🛡️ SOC (Security Operations Center) Analyst Roadmap 💻🚀
SOC Analysts are the frontline defenders who monitor, detect, investigate, and respond to cybersecurity threats in real time 🔥
####🔹 What is a SOC Analyst?
- 🚨 Monitors security alerts and events
- 🔍 Investigates suspicious activities
- 🛡️ Detects and responds to cyber threats
- 📊 Analyzes logs from multiple security tools
- 🚑 Supports incident response activities

  #### 🔹 🔹 Common Interview Questions
1. 🧠 What is a SIEM?
- → A platform that collects and correlates security events from multiple sources.
2. 🧠 What is an IOC?
- → Indicator of Compromise that suggests malicious activity.
3. 🧠 Difference between IDS and IPS?
- → IDS detects threats; IPS detects and blocks threats.
4. 🧠 What is a False Positive?
- → A security alert incorrectly identified as malicious.
